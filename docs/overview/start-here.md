# Start Here

If you are new to this repository, take this path:

1. Read [strongest-results.md](strongest-results.md).
2. Read [../theorems/proof_status.md](../theorems/proof_status.md).
3. Run `python scripts/validate/run_research_checks.py`.
4. Open the generated symbolic report in `data/generated/validation/symbolic_closure_report.json`.
5. Only then move into the broader toolkit commands.

This order keeps the theory/core results separate from the solver tooling and the optimizer bridge.
