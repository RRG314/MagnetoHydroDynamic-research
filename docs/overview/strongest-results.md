# Strongest Results

## 1. Broad exact cylindrical families under constant resistivity

The repository now supports a stronger statement than the earlier power-family memo alone.

Exact under constant `eta`:
- `alpha = f(r)`, `beta = r*theta`
- `alpha = f(r)`, `beta = z`
- `alpha = r*theta`, `beta = g(z)`

## 2. A clean conditional family

For `alpha = f(r)`, `beta = theta`, exactness holds only when `f(r)` is quadratic.

That turns an earlier exploratory case into a clearer classification result.

## 3. Variable resistivity is the main current obstruction

Once `eta` varies with `r`, the smooth exact families above are generically broken in the tested cylindrical setting.

That is the clearest present limit of the program and the main mathematically serious next direction.
