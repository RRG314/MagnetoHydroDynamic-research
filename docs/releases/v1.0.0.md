This release marks the first public version of the repository in its current research-program form.

It shifts the project away from a loose toolkit presentation and into a clearer theory-first structure centered on closure questions, symbolic verification, explicit limits, and reproducible checks.

## Why this release matters

- The strongest exact and symbolically verified results are now easy to find.
- The variable-resistivity obstruction is surfaced clearly instead of being buried in notes.
- The repository now separates core closure theory from the Topological Adam bridge work.
- Reproducible validation and proof-status documents are part of the main path.

## Included in v1.0.0

### 1. A theory-first repository structure

- start-here and strongest-results documents
- proof-status map and derivation notes
- canonical discovery documents
- final report and file index for orientation

### 2. Stronger symbolic closure program

- reusable symbolic closure module
- canonical symbolic report output
- CLI research entry point for symbolic checks
- symbolic regression tests for exact and obstruction cases

### 3. Clearer mathematical picture

Strongest current results include:
- exact cylindrical families under constant resistivity
- the conditional exactness result for `alpha = f(r), beta = theta`
- explicit obstruction of broad smooth exact families under variable `eta(r)`

### 4. Controlled cross-repository bridge

- the relationship to Topological Adam is retained
- it is now clearly presented as a secondary applied branch rather than the main identity of this repository

## Honest scope

This release does not claim to be a finished closure theory for general resistive MHD.

What it does claim:
- the cylindrical exact-family picture is materially clearer than before
- the variable-resistivity obstruction is now part of the main public story
- the symbolic research layer is reproducible and test-backed

What remains open:
- perturbative closure corrections for slowly varying `eta(r)`
- broader exact-family criteria beyond the current cylindrical families
- stronger numerical ties between symbolic obstruction formulas and simulation behavior
