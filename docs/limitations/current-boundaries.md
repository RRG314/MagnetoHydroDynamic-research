# Current Boundaries

## 1. Variable resistivity is not solved

The repository now makes this much clearer. Nonconstant `eta(r)` is the main present obstruction to smooth exact closure in the strongest symbolic families.

## 2. Toroidal work is still exploratory

The toroidal experiments are useful, but they are not yet on the same footing as the cylindrical derivations.

## 3. The numerical toolkit is baseline-quality

The solver and closure sandbox are useful for controlled experiments, not production-grade MHD simulation claims.

## 4. The Topological Adam bridge is secondary

It remains worth documenting, but it should not be allowed to blur the theory/core identity of this repository.
