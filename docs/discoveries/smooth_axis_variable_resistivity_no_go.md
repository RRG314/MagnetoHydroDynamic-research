# Smooth Axis Variable-Resistivity No-Go

## Main statement

On the supported cylindrical radial families with nonconstant `eta(r)` on an interval touching the axis, there are no nonconstant smooth exact survivors.

## Supported families

### Family 1
For `alpha=f(r)`, `beta=r*theta`, exactness requires

$$
2 r f''(r) + f'(r) = 0,
$$

so

$$
f(r)=a\sqrt{r}+b.
$$

This is exact on annuli but not smooth at `r=0` unless `a=0`.

### Family 2
For `alpha=f(r)`, `beta=z`, exactness requires

$$
r f''(r)+f'(r)=0,
$$

so

$$
f(r)=a\log r+b.
$$

This is singular at the axis unless `a=0`.

## Consequence

Within these supported radial families:
- annular exact survivors exist
- smooth axis-touching nonconstant exact survivors do not

## Honest scope

This is not yet a complete theorem for every conceivable cylindrical ansatz. It is a clean no-go on the currently strongest supported variable-resistivity radial lanes.
